### FitLife

## Описание.
 Проект FitLife — это MVP-версия приложения, которое помогает пользователям отслеживать показатели здоровья, такие как индекс массы тела (ИМТ) и рекомендуемая норма потребления воды.

## Как работает программа.
 Программа запрашивает у пользователя имя, возраст, вес и рост. На основе этих данных она рассчитывает ИМТ и рекомендуемую норму потребления воды в день. Результаты выводятся в удобном для чтения формате.

## Для развёртывания проекта локально вам нужно выполнить несколько шагов.

# Установите зависимость.
 Убедитесь, что у вас установлен Python и виртуальное окружение (в данном проекте - venv). Затем активируйте виртуальное окружение и установите зависимость, в проекте есть файл "requirements.txt".

# Запустите проект.
 После установки зависимости Вы можете запустить скрипт. Основной скрипт называется "fitlife.py".

# Проверьте работу программы.
 Введите необходимые данные (имя, возраст, вес, рост), чтобы убедиться, что программа работает корректно и выводит ожидаемые результаты.

## Ключевые переменные и константы.
 - WATER_PER_KG: стандартная рекомендация потребления воды на килограмм веса (30 мл/кг). -
 - ML_PER_L: количество миллилитров в литре (1000 мл). 
 - FLINE: строка из 40 дефисов для визуального разделения вывода.
